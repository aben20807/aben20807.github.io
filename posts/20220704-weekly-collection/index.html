    Vim 9.0 發布

    跨平台開發套件

    原先一般會用 Electron (GitHub 開發的開源框架4)，但是缺點太多 (檔案過大、記憶體使用量過多)，主因是使用 Node.js 當作後端，且前端使用 Chromium，有名的使用案例是 Atom editor (可惜 GitHub 年底要收掉它了)。Tauri 將後端轉成 Rust，前端為自行開發的輕量框架 (TAO, WRY)，大幅減少相關資源的使用量。

    • 官網: https://tauri.app/
    • GitHub: https://github.com/tauri-apps/tauri
