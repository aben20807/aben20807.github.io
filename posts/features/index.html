    Copy code block

    右上角可以複製程式碼，目前會自動忽略 prompt (每行開頭的 $, >)。加上 {copy=no} 可以關閉複製按鈕。

    ```bash
$ echo "hello"
```

```bash {copy=no}
$ echo "hello"
```
