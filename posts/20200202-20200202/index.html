    想當年，高中程式設計上機考最後一題就是迴文，當時能帶大抄，剛好有迴文所以就直接貼上了， 記得打了一堆自己不太懂的程式碼。當時我左邊的同學霸氣不帶大抄，雖然他最後一題沒寫出來，不過卻令我相當佩服呢。

    啊，完蛋，是字串反轉還是迴文判斷啊@@，算了，小時候的事了(x

    因應時節(?)，所以寫了一個簡單的迴文檢測程式。

    #include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main(int argc, char *argv[]) {
    char *s = strdup(argv[1]);
    int len = strlen(s);
    int mid = len / 2;
    for (int i = 0; i < mid; i++) {
        if (s[i] != s[len - 1 - i]) {
            printf("Not Palindrome QuQ\n");
            goto exit;
        }
    }
    printf("Palindrome! OuO\n");
exit:
    free(s);
    return 0;
}

    $ gcc -o palindrome palindrome.c
$ ./palindrome 20200202
Palindrome! OuO

$ ./palindrome 20200203
Not Palindrome QuQ
